"Continuation-Derived Relational Initialization for Goal-Conditioned Critics in Sparse-Reward Aerial Reinforcement Learning 1. 연구 배경과 문제의식 드론 자율비행을 위한 sparse-reward goal-conditioned 강화학습에서 가장 널리 알려진 어려움은 데이터 효율성이다. 실제 비행 데이터는 획득 비용이 높고 위험하며, 시뮬레이션에서도 초기 성공 궤적은 드물다. 그래서 기존 연구는 대체로 두 방향으로 문제를 다뤄 왔다. 하나는 HER, reward shaping, demonstration, model-based augmentation처럼 더 많은 유효 학습 신호를 확보하는 방향이고, 다른 하나는 curriculum, exploration control처럼 성공 궤적이 더 빨리 나타나게 만드는 방향이다. 본 연구는 이 흐름을 부정하지 않는다. 다만 그보다 한 단계 더 안쪽의 질문을 묻는다. 초기 sparse-reward 구간에서, goal-conditioned critic은 같은 상태 맥락 안에서 여러 목표 후보 중 무엇을 먼저 더 유망하게 보아야 하는지를 충분히 잘 구분하고 있는가? 이 질문이 중요한 이유는, sparse reward와 poor coverage는 이미 알려진 문제이지만 그것만으로는 critic 내부에서 어떤 종류의 초기 선택 오류가 먼저 드러나는지까지는 충분히 설명하지 못하기 때문이다. goal-conditioned critic은 절대적인 return 규모를 정확히 맞추기 훨씬 이전부터, 고정된 상태 맥락 아래 candidate goals 사이의 상대적 선호 구조를 어느 정도 형성해야 한다. 그렇지 않으면 초기 단계에서 더 직접적이고 덜 위험한 short-horizon branch를 놓치고, 덜 적절한 goal candidate를 상대적으로 더 선호하는 일이 생길 수 있다. 본 논문은 바로 이 지점을 겨냥한다. 즉 “데이터가 적다”는 일반론을 반복하는 것이 아니라, 데이터가 적을 때 critic 내부에서 특히 먼저 드러날 수 있는 same-context goal choice quality의 부족을 operational하게 분리하고, 그것을 직접 겨냥하는 initialization principle을 제안한다. 2. 논문이 다루는 중심 개념 본 논문이 다루는 중심 개념은 다음과 같다. early-stage same-context choice misalignment 고정된 state context 아래 candidate goal 집합이 주어졌을 때, critic-derived goal preference가 초기 학습 단계에서 externally tested short-horizon branch quality와 충분히 맞지 않는 상태. 이 개념의 지위는 의도적으로 조심스럽게 둔다. 본 논문은 이것을 새로운 pathology라고 주장하지 않는다. 강화학습의 본질적 bottleneck이라고 단정하지도 않는다. goal-conditioned critic의 일반적 결함을 발견했다고 말하지도 않는다. 대신 본 논문은 다음 정도를 주장한다. 초기 sparse-reward 구간에서는 same-context goal ordering이 충분히 형성되지 않을 수 있다. 이 현상은 단순한 정렬 점수 차이를 넘어 실제 short-horizon branch choice quality와 연결될 수 있다. trajectory continuation에서 얻은 제한적 relational evidence로 critic-side encoder를 warm-start하면, 이 부분을 더 직접적으로 보완할 수 있다. 그 효과는 generic self-supervised learning이나 temporal-only prior만으로는 충분히 설명되지 않을 수 있다. 즉 본 논문은 “새로운 실패 모드를 발견했다”는 논문이 아니라, 초기 선택 구조의 특정 단면을 따로 보면 실용적으로 의미가 있으며, 이를 겨냥한 초기화가 실제 학습에 유용한지 검증하는 논문이다. 3. 왜 이 framing이 필요한가 “초기 critic이 불안정하다”는 설명은 너무 넓다. 그 설명 아래에서는 HER, exploration bonus, model-based augmentation, contrastive learning, value regression, progress prediction 등 너무 많은 개입이 동시에 타당해 보인다. 반면 본 논문이 겨냥하는 것은 더 구체적이다. 초기 sparse-reward 구간에서 critic이 같은 상태 맥락 아래 여러 candidate goals 중 무엇이 더 직접적이고 덜 위험한 short-horizon branch인지를 충분히 잘 구분하지 못할 수 있으며, 이 문제를 줄이려면 absolute value calibration 이전에 same-context relational ordering을 더 잘 잡아주는 warm-start가 필요하다는 것이다. 따라서 본 논문에서 Kendall τ 같은 alignment score 자체는 목적이 아니다. 더 중요한 질문은 다음이다. critic이 top-1로 선호한 goal이 실제 short-horizon rollout에서 첫 feasible approach를 얼마나 자주 놓치는가? 같은 calibration 수준에서도 alignment 차이가 초기 crash-prone branching decision 차이로 이어지는가? 같은 상태 맥락에서 goal preference ordering 차이가 실제 short-horizon branch quality 차이와 연결되는가? 즉 본 논문은 “외부 evaluator에 잘 맞는 표현을 만든다”가 아니라, 초기 잘못된 goal branch 선택 구조를 줄이는 critic-side warm-start를 제안하는 논문이어야 한다. 4. 핵심 가설 본 논문은 아래 다섯 가지 예측을 갖는다. 초기 sparse-reward replay에서는 same-context goal ordering이 short-horizon feasible choice와 잘 맞지 않는 구간이 존재한다. 이 misalignment는 generic self-supervised learning보다 same-context relational warm-start로 더 직접적으로 개선된다. 이 차이는 단순 temporal closeness prior만으로는 설명되지 않는다. terminal position alone으로 local branch quality가 잘 닫히지 않는 aerial task일수록 그 차이가 더 커진다. 이러한 alignment 및 choice improvement는 early success, crash reduction, first-feasible-approach selection과 연결된다. 이 중 논문의 성패를 가르는 것은 특히 2번, 3번, 5번이다. 5. 이 논문이 아닌 것 본 논문은 다음이 아니다. 새로운 강화학습 알고리즘 reward shaping HER 규칙 변경 exploration method value regression pretraining ordering 기준으로 warm-start하는 initialization principle 6. 문제 측 조건과 방법 측 신호를 분리한다 초기 sparse-reward 구간을 다룰 때는 문제 자체와 제안 방법이 실제로 활용 가능한 신호를 분리하는 것이 중요하다. 6.1 Problem-side support poverty 문제 측의 support
